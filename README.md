# Mega Coocook Excel

Convert Coocook shopping list PDFs into structured and customizable Excel workbooks.

## Features (planned)

- Read Coocook shopping list PDFs
- Export to Excel
- Automatic categorization
- Dashboard
- Shopping lists by category
- Shopping lists by day
