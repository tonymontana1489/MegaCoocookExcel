# Sprint 3.6

## Ziel

Ein Launcher für alle Entwicklungsumgebungen.

### Desktop-PC

- startet PyWebView

### GitHub Codespaces

- erkennt fehlende GUI
- startet automatisch einen HTTP-Server
- öffnet das Dashboard im Browser

## Start

```bash
python -m megacoocookexcel.dashboard.launcher
```

Es ist kein separater `python -m http.server` mehr nötig.
